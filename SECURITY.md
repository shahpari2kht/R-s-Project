# Security Policy | سیاست امنیتی

## Reporting a Vulnerability
If you discover a security vulnerability, please report it via email to: **shahpari2kht@gmail.com**  
We will respond within 48 hours and acknowledge the reporter.

## گزارش آسیب‌پذیری
در صورت یافتن مشکل امنیتی، لطفاً از طریق ایمیل به **shahpari2kht@gmail.com** اطلاع دهید.  
ما ظرف ۴۸ ساعت پاسخ خواهیم داد و گزارش‌دهنده را معرفی خواهیم کرد.
