# R-Shiny-MTCARS-Dashboard | داشبورد MTCARS با Shiny

این پروژه یک داشبورد تعاملی ساده با داده `mtcars` و پکیج **Shiny** در R ایجاد می‌کند.

This project creates a simple interactive dashboard for the `mtcars` dataset using **Shiny** in R.

---

## امکانات | Features
- فیلتر داده بر اساس مصرف سوخت، تعداد دنده و نوع موتور  
- مشاهده جدول و نمودار براساس فیلتر کاربر

- Filter data by fuel consumption, number of gears, and engine type  
- View table and plots based on user-selected filters

---
