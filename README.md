# R-s-Project

**My R Projects | مجموعه پروژه‌های R**  

این ریپازیتوری شامل مجموعه‌ای از پروژه‌های مهم و آموزشی با زبان **R** است. هر پروژه شامل کد قابل اجرا، داده‌های نمونه و توضیحات قدم‌به‌قدم است. هدف: نمایش مهارت‌های **Data Science، تحلیل داده‌ها و توسعه اپلیکیشن Shiny**.  

This repository contains several key and educational **R projects**. Each project includes executable code, sample datasets, and step-by-step instructions. Purpose: showcase **Data Science skills, data analysis, and Shiny app development**.

---

## فهرست پروژه‌ها | Projects Included

### 1️⃣ R-Marketing-Analysis
- تحلیل داده‌های بازاریابی مستقیم با **R** و گرافیک‌های **ggplot2**  
- Direct marketing data analysis using **R** and **ggplot2** visualizations  

### 2️⃣ R-Shiny-MTCARS-Dashboard
- اپلیکیشن داشبورد تعاملی برای داده `mtcars` با پکیج **Shiny**  
- Interactive dashboard application for the `mtcars` dataset using **Shiny**

---

## راهنمای اجرا | How to Run

1. به فولدر پروژه مورد نظر مراجعه کنید.  
2. توضیحات هر پروژه در فایل `README.md` مخصوص آن پروژه موجود است.  
3. برای اجرای پروژه‌ها به **R** و (در صورت نیاز) **RStudio** نیاز دارید.  
4. بسته‌های لازم (Dependencies) را طبق دستورالعمل README داخلی نصب کنید.

Navigate to the folder of the desired project.  
Each project includes its own `README.md` with instructions.  
**R** and optionally **RStudio** are required.  
Install dependencies as specified in each internal README.

## Requirements | پیش‌نیازها
- R (version ≥ 4.0)  
- RStudio (optional)  
- Packages: `ggplot2`, `shiny`, `dplyr`, `tidyr`, ... (see each project README)
