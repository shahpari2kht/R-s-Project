# R-Marketing-Analysis | تحلیل بازاریابی با R

این پروژه شامل تحلیل داده‌های بازاریابی مستقیم با استفاده از **R** است.  
تحلیل با **RMarkdown** انجام شده و نمودارهای توصیفی مختلف رسم شده‌اند.

This project contains a direct marketing data analysis using **R**.  
The analysis is performed with **RMarkdown** and includes various descriptive visualizations.

---

## فایل‌ها | Files
- `MarketingAnalysis.Rmd` – فایل نوت‌بوک با کد و گزارش  
- `DirectMarketing.csv` – دیتاست پروژه
